# django-microblog-app
Djangoを使った画像投稿のできるブログサイトを作成。    
Herokuにデプロイしており下記のURLからアクセスできます。  
※起動に少し時間がかかります。  
https://django-microblog-app.herokuapp.com/

> 実装した機能
>* 記事のCRUD処理
>* DB接続（local:SQLite / Heroku:PostgreSQL）
>* 動画投稿（クラウドストレージに保存）
>* フラッシュメッセージ
>* ページネーション

## Dependency
Python 3.6 / Django 2.2.3 / Cloudinary
